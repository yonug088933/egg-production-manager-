# 🥚 계란 선별포장 생산 관리 시스템 v5

본점(식자재) / 지점(제품) 이원화 + **역학조사 추적 시스템** 통합 버전

---

## ✨ 주요 기능

### 🏢 본점 (식자재) - 판수→파렛 자동환산
- 등급(왕/특/대/중) × 판수(5~1판) 조합 20종 기본 제공
- 판수 입력 시 파렛 자동 계산 (예: 특5판 120판 → 2.00 PLT)

### 🏪 지점 (제품) - 박스→개수 자동계산
- 청정원, CJ, 하림 등 브랜드별 제품 관리
- 박스 입력 시 개수 자동 계산 (예: 동물복지 25구 9박스 → 81개)

### 🚚 출고 (모하지 3PL) + 역학조사 차량 정보
- 본점: 판수→파렛 자동환산
- 지점: 박스→개수 자동계산
- **차량번호 + 운전자** 기록 (역학조사 필수)

### 🛡️ 역학조사 추적 시스템 (NEW!)
- **농장 마스터 등록**: 한 번 등록 후 입고 시 선택만
- **Lot 번호 자동 부여**: 입고 1건 = 1 Lot (예: L-260425-001)
- **난각 표시 + 차량번호** 필수 기록
- **생산 시 사용 Lot 선택**: 어떤 입고분이 어느 작업에 쓰였는지 추적
- **역학조사 검색**: 농장명/Lot번호/날짜로 입고→생산→출고 즉시 추적
- **추적 보고서 CSV 다운로드**: 검사 시 바로 제출 가능

### ☁️ Firebase 실시간 동기화
- 모든 기기에서 실시간 공유
- 사무실 PC·현장 핸드폰·대표님 스마트폰 모두 동일 데이터

## 🛡️ 역학조사 시스템 사용법

### 1단계: 농장 등록 (한 번만)
입고 화면 → 농장 선택 옆 ⚙️ 버튼 → 새 농장 등록
- 농장명, 농장번호(난각용), 등록번호, 농장주, 연락처, 주소

### 2단계: 입고 시 매번 기록
- ✅ 농장 선택 (등록된 곳에서 자동 채워짐)
- ✅ **난각 표시** (예: `1206 M3FDS 2`)
- ✅ **차량번호** (예: `12가 3456`)
- ✅ 수량(판수)
- → 저장하면 **Lot 번호 자동 부여** (L-260425-001)

### 3단계: 생산 시 사용 Lot 선택
- 작업 시작 시 "사용한 원란 Lot" 체크박스에서 선택
- 여러 Lot 동시 선택 가능 (Lot 섞어서 작업 시)

### 4단계: 출고 시 차량 기록
- 출고 차량번호 + 운전자 기록

### 🚨 역학조사 발생 시
1. 헤더 우측 빨간 **[역학조사]** 버튼 클릭
2. 검색 방식 선택: 
   - **Lot 번호로** (가장 정확)
   - **농장명으로** (특정 농장 지목 시)
   - **날짜 기간으로** (특정 기간 입고분 전체 추적)
3. 결과 확인:
   - ✅ 해당 입고 기록 (Lot/농장/난각/차량/수량)
   - ✅ 사용된 생산 기록 (날짜/시간/품목/수량)
   - ✅ 관련 출고 기록 (생산일 ±7일 내, 거래처/차량/수량)
4. **CSV 보고서 다운로드** → 즉시 제출 가능

## 🚀 설치 & 실행

### 1. 의존성 설치
```bash
npm install
```

### 2. Firebase 설정
`src/App.jsx` 상단의 `firebaseConfig`에 본인 Firebase 정보 입력

### 3. Firestore 보안 규칙
Firebase 콘솔 → Firestore → 규칙:
```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
```

### 4. 실행
```bash
npm run dev      # 로컬 실행
npm run build    # 배포용 빌드
```
